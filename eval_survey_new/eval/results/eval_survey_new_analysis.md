## 6. 关键发现

1. **reasflow 在 Dim1（内容准确性）上优势显著**：均分 9.26 vs 6.33（+2.92）。
  - 原因：reasflow 的引用来自 AutoSurvey 339 篇论文库，检索时必然命中，获得 arxiv_id 后可做全文验证，通常评为 L3（基本无误）。
  - base 的引用来自 LLM 知识库，多数不在 339 篇库中，检索无结果后被判定为「无法对应真实论文」（L0，视为捏造），得 0 分。
2. **Dim2（引用相关性）也有正向差异**：均分 6.67 vs 6.33（+0.33）。
  - Dim1 中被判定为捏造的引用（L0）传递给 Dim2，直接计为「不相关」(score=0)，并从 Key Ref 覆盖和引用计数中剔除。
  - 这一级联效应使 base 的有效引用数量减少，覆盖率和质量分都受到影响。
3. **总分优势 +3.26**：reasflow 在 3 个任务 × 3 个评测模型 × 4 个 backbone 的 36 组配对比较中，胜率达 **77.8%**（28胜 8负）。
4. **不同 backbone 的差异**：
  - dpsk backbone: reasflow 优势最大（+5.42），因为 dpsk 生成的 base 引用缺少 arxiv_id 比例最高
  - sonnet4.6 backbone: reasflow 优势最小（+0.45），因为 sonnet4.6 生成的 base 引用质量较高，部分自带 arxiv_id
